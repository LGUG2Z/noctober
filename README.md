# Noctober

> A desktop application that can send your Kobo highlights to [Notado](https://notado.app) in a couple of clicks

## What is Noctober?

A fork of the excellent [October](https://github.com/marcus-crane/october) by [Marcus Crane](https://utf9k.net) which sends your Kobo highlights to Notado instead of Readwise.

You can learn more by browsing the official [October documentation website](https://october.utf9k.net).
